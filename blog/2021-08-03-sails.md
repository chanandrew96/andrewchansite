經npm安裝sails
``` npm install sails -g ```
建立一個新專案，這次先利用預設的專案
``` sails new [project-name] ```
然後選擇(1)  
當見到``` info: Created a new Sails app `test-project`! ```便代表專案建立成功  
使用``` sails lift ```來開啟Sails Web Application  

[Reference](https://sailsjs.com/get-started)
